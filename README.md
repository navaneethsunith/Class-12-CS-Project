# Product Management System
Class 12 CBSE Computer Science Project (2025-26)

## Overview
A Python and MySQL-based application for managing supermarket inventory. The system replaces manual record-keeping with a digital database to track stock and monitor item expiration.

## Features
* Secure Login: Access control via admin password.
* Inventory Management: Add new products and delete existing records.
* Stock View: Display all current products, prices, and quantities.
* Expiry Tracking: Automated check for products past their expiry date.

## Technical Stack
* Language: Python 3.x
* Database: MySQL
* Modules: mysql.connector, datetime

## System Requirements
* OS: Windows / macOS / Linux
* Dependencies: MySQL Server, mysql-connector-python

## Database Schema
Table: Products
- ProductID (Primary Key)
- ProductName
- Price
- Quantity
- ExpiryDate
